# Stopwatch

A simple Javascript stopwatch, written in es2015, using webpack.

<a href="https://stopwatch.oskarnijs.nl/">Demo</a>

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
